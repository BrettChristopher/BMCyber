NBA Active Directory IAM Project

A full Identity & Access Management (IAM) simulation using Microsoft Active Directory to model identity governance for an NBA organization.

Environment Overview

  • Domain: nba.local
  • Domain Controller: DC01
  • File Server: FS01
  • Application Server: APP01
  • OS: Windows Server 2022

Active Directory Architecture

  • Players
  • Coaches
  • Analytics
  • Medical
  • FrontOffice
  • IT
  • PrivilegedRoles
  • DisabledUsers

RBAC & Access Controls

  • GRP_Players: Player portal
  • GRP_Coaches: Playbooks
  • GRP_Analytics: Dashboards
  • GRP_Medical: PHI
  • GRP_FrontOffice: Contracts
  • GRP_ITAdmins: Admin privileges

Privileged Access Model

Privileged Access Model
Tier 0 – Domain Admins • Full domain control • Protected accounts Tier 1 – Server Admins • Manage FS01, APP01, DC01 • No domain-wide rights Tier 2 – Helpdesk • Password resets • Basic user support JIT Elevation • Temporary membership in GRP_ITAdmins • Approval required
